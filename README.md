# Easy Dev Project

This is a simple and clean project created for learning, practice, and contribution tracking on GitHub.

## What This Project Contains
- Basic frontend structure (HTML, CSS, JS)
- Simple backend logic (Python)
- Configuration file (JSON)
- Notes and test files

## How to Use
1. Open `index.html` in your browser
2. Run `app.py` using Python
3. Edit files and commit changes to track progress

## Goal
Practice daily updates and improve development skills step by step.
